## 🚀 주요 기능
* **자동 대화 스킵:** 이미지 인식을 통해 대화창과 선택지를 자동으로 클릭합니다.
* **인연 스토리 연속 진행:** 메시지 목록에서 읽지 않은 스토리를 순차적으로 탐색합니다.
* **모든 메시지 확인:** 안 읽은 메시지를 모두 읽을 때 까지 프로그램은 끝나지 않습니다.
<br>

## 📦 다운로드 및 사용 방법
1. [Releases](https://github.com/ChoHye0N/Auto-Affection-Skip/releases) 페이지에서 최신 버전의 `.zip` 파일을 다운로드합니다.
2. 압축을 해제한 뒤 `AutoAffectionSkip.exe`를 실행합니다.
3. 블루 아카이브 화면을 띄운 상태에서 '시작' 버튼을 누릅니다.
4. 오류가 발생하거나 매크로를 중단하려면 '종료' 버튼을 누릅니다.
<br>

## 🚧 주의 사항
* 본 프로그램은 이미지 인식 기반으로 작동하며 게임 메모리를 직접 수정하지 않습니다.
* 전체화면(1920x1080)을 적극 권장하며, 창모드와 다른 해상도 지원은 추후에 업데이트할 예정입니다.
